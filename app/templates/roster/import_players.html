CSV Format Requirements

Your CSV file should include the following columns:

  • Player name (required) - Full player name
  • Age (optional) - Current age as reference
  • DOB (recommended) - Date of birth in DD/MM/YYYY format
  • General Position - RHP, LHP, CATCHER, OUTFIELDER, INFIELDER
  • Bats - LEFT, RIGHT, SWITCH
  • Throws - LEFT, RIGHT
  • Specific POS - C, 1B, 2B, 3B, SS, LF, CF, RF, RHP, etc.
  • Birthplace - Format: CITY - STATE (e.g., VALENCIA - CAR)
  • Actual League - Current league (MLB, AAA, etc.)
  • Team - External team name
